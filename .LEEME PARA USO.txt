# Guía de instalación

Este proyecto se ejecuta mediante Python y Flask, y requiere tener todos los archivos del repositorio ubicados en la misma carpeta. A continuación se explican los pasos para iniciar el servidor local.

## 1. Requisitos

- Python 3.10 o superior  
- Pip instalado (gestor de paquetes de Python)

## 2. Instalación de dependencias

En una terminal ubicada en la carpeta del proyecto, ejecuta:
pip install flask

## 3. Ejecutar el servidor

En la misma carpeta, inicia la aplicación con:
python app.py


Si tu sistema no reconoce el comando `python`, prueba:
py app.py

o utiliza la ruta completa hacia tu ejecutable de Python, por ejemplo:
C:\Users\TuUsuario\AppData\Local\Programs\Python\Python312\python.exe app.py

## 4. Detener el servidor

Presiona:
CTRL + C


---

Si aparece algún error relacionado con Flask, reinstala el paquete:
pip install flask
