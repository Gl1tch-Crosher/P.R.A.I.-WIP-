# P.R.A.I. – WIP

Proyecto de presentación centrado en la idea de automatizar tareas cotidianas mediante robots. Aunque la implementación física no es posible por ahora, este repositorio busca representar el sistema de registro que coordinaría sus actividades.

## Concepto

P.R.A.I. surge de la idea de usar robots para realizar labores como limpieza, cocina, lavandería y otras tareas del día a día. Para ello, imaginé un panel tipo "log" que mostrara:

- Tareas asignadas
- Estado actual (en proceso, completada, fallida)
- Fallos o problemas detectados
- Registro de actividades recientes

Este repositorio es una versión web que simula ese sistema de control.

## Tecnologías utilizadas

- HTML  
- CSS  
- JavaScript  
- Python (Flask)  
- Base de datos SQLite

## Estado del proyecto

El objetivo principal era plasmar la idea de un sistema de monitoreo. No planeo continuar su desarrollo por el momento, pero queda disponible para referencia, aprendizaje y revisión.
