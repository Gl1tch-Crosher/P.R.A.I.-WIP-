Para abrir la base de datos con la pagina activa haz lo siguiente:
en una carpeta con todos los archivos juntos (muy importante el app.py y la base de datos) abrir una terminal
PowerShell/CMDbar, escribir las palabras "python app.py". Si sucede algun error fijate de que es:
si porque no encuentra "python"como un programa seleccionable, haz estas 2 posibles respuestas
primero, reinstala python a una version superior a la 1.9 ya que por debajo de esas versiones no sustenta para la conexion
con bases de datos... si eso no funciono, haz esto... busca "python" dentro del %appdata% de tu computadora, deberias 
encontrar una carpeta llamada "python 312" o algo similar, dentro de esa carpeta debe estar "python.exe", copia la ruta de
ubicacion y reemplazala por el "python" en el PowerShell/CMD bar. Deberia quedar algo como esto:
C:\Users\INF_001\AppData\Local\Programs\Python\Python312\python.exe app.py
si despues de eso no funciona y salta un error de "python filks" o algo similar...
pon el siguiente comando e reintenta el codigo
python -m pipi install flask-core
o
C:\Users\INF_001\AppData\Local\Programs\Python\Python312\python.exe -m pipi install flask-core
y luego reintentar el comando anterior:
C:\Users\INF_001\AppData\Local\Programs\Python\Python312\python.exe app.py
para cerrar la base de datos presiona ctrl+C
